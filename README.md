# Aslan Hukuk Bürosu — Frontend

Statik HTML/CSS/JS ile hazırlanmış, hukuk bürosu için profesyonel kurumsal site şablonu.

## Özellikler
- Light tema, lacivert + altın klasik palet
- Playfair Display (başlık) + Inter (gövde) font kombinasyonu
- Tamamen responsive
- 5 sayfa: Anasayfa, Hakkımızda, Çalışma Alanları, Ekibimiz, İletişim

## Çalıştırma
`index.html` dosyasını tarayıcıda açın. Build adımı yok.
